hed

APT & YUM repository for hed, a small plugin-based terminal editor written in C.

Debian / Ubuntu (APT)

curl -fsSL https://42dotmk.github.io/hed/hed-archive-keyring.asc \
  | sudo gpg --dearmor -o /usr/share/keyrings/hed.gpg
echo "deb [signed-by=/usr/share/keyrings/hed.gpg] https://42dotmk.github.io/hed/deb ./" \
  | sudo tee /etc/apt/sources.list.d/hed.list
sudo apt update && sudo apt install hed

Fedora / RHEL (DNF)

sudo tee /etc/yum.repos.d/hed.repo <<'EOF'
[hed]
name=hed
baseurl=https://42dotmk.github.io/hed/rpm
enabled=1
gpgcheck=1
gpgkey=https://42dotmk.github.io/hed/hed-archive-keyring.asc
EOF
sudo dnf install hed

If the repository is published unsigned (no signing key configured), use [trusted=yes] for APT and gpgcheck=0 for DNF.

Other install paths — one-line installer, Homebrew tap, Arch (AUR), Void — are in the project README.
